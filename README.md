Sheet
=====

WPF  logic  drawing  application.

Keyboard  Shortcuts:

    Ctrl+O          Open
    Ctrls+S         Save
    N               Mode None
    S               Mode Selection
    L               Mode Line
    R               Mode Rectangle
    T               Mode Text
    I               Mode Signal
    A               Mode AndGate
    O               Mode OrGate
    Ctrl+Z          Undo
    Ctrl+Y          Redo
    Ctrl+X          Cut
    Ctrl+C          Copy
    Ctrl+V          Paste
    Del             Delete
    Ctrl+Del        Reset
    Ctrl+A          Select All
    B               Create Block
    Ctrl+B          Break Block
    Up              Move Up
    Down            Move Down
    Left            Move Left
    Right           Move Right
    H               Show Help

Mouse:

    Left Click            Create / (+Drag) Move / (+Drag) Selection / Select / Deselect
    Right Click           Pan / Cancel Create / Cancel Selection / Deselect / Edit Text
    Mouse Wheel           Zoom
    Double Middle Click   Reset Pan and Zoom
