Sheet
=====

Diagram drawing application.

Keyboard  Shortcuts:

    (File)
    Ctrl+Shift+N    New Solution
    Ctrl+Shift+O    Open Solution
    Ctrl+Shift+S    Save Solution
    Ctrl+Shift+C    Close Solution
    Ctrl+O          Open
    Ctrl+S          Save
    Ctrl+E          Export
    Ctrl+L          Library
    Ctrl+D          Database
    Alt+F4          Exit
    (Edit)
    Ctrl+Z          Undo
    Ctrl+Y          Redo
    Ctrl+X          Cut
    Ctrl+C          Copy
    Ctrl+V          Paste
    Del             Delete
    Ctrl+Del        Reset
    Ctrl+A          Select All
    B               Create Block
    Ctrl+B          Break Block
    Up              Move Up
    Down            Move Down
    Left            Move Left
    Right           Move Right
    F               Toggle Fill
    (View)
    Ctrl+0          Zoom to Page Level
    Ctrl+1          Actual Size
    (Mode)
    N               Mode None
    S               Mode Selection
    I               Mode Insert
    P               Mode Point
    L               Mode Line
    R               Mode Rectangle
    E               Mode Ellipse
    T               Mode Text
    D               Mode Image
    (Logic)
    Q               Invert Line Start
    W               Invert Line End

Mouse:

    Left Click            Create / (Hold+Drag) Move / (Hold+Drag) Selection / Select / Deselect
                          Select/deselect single/multiple items while Ctrl key is pressesed.
    Right Click           Pan / Cancel Create / Cancel Selection / Deselect / Edit Text
    Mouse Wheel           Zoom
    Double Middle Click   Auto-Fit (+Ctrl key to Reset Pan and Zoom)
    Mouse Over (+Shift)   Select signle item under mouse cursor.
